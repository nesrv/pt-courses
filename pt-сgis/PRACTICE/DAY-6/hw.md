Сигнатура функции, следующая:

```python
def get_menu(s): ...
```
Определите декоратор для этой функции с именем `show_menu`, который отображает список на экран в формате:

```
1. Пункт_1
2. Пункт_1
...
N. Пункт_N
```

Примените декоратор `show_menu` к функции `get_menu`, используя оператор @.

```python
def show_menu(func):
   ...

@show_menu
...
```
